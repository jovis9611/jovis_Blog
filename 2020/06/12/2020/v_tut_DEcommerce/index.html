Project Set Up

  1. Clone these from these https://github.com/justdjango/django_project_boilerplate and https://mdbootstrap.com/freebies/jquery/e-commerce/ in visual studio

  2. Run pip install virtualenv at the cloned folder.

    • Then runvenv env and env\scripts\activate
    • Will come out Python virtual environment console.
    • You can install Python library and or run Python cmd in this environment.

  3. django cmd

    • python manage.py runserver is to deploy webpage on browser.
    • python manage.py createsuperuser for manage the admin panel.
    • python manage.py + makemigrations or migrate cmd are use after doing modification in models.py.

  4. Sample Django forlder structure:

  • Sub app folder(url,view,admin,model).py
    • app. Difference app can integrated with main app.
  • Main app folder > setting >Installed app: >url
    • main app
  • static folder
    • Have to modify static folder url in setting for execute js,css in static folder.
  • template (html)
    • This folder is to store .html files
    • Inheritance of html. Base html and other html
    • It has like {extends 'base.html'} or {include '.html'}
    • extends (Inherited whole compnent)
    • includes (insert other component)
    • It also has {load xx}. It function like import

  1. Files nside Django main app and sub app:

    • url. (for link)
    • view (return object to template)
    • model (for database modification)
    • admin (It has related to the model and it for register database into admin panel also)

  2. Move folders (static folders) from mdbootstrap to static folder. Move all html files from mbdbootstrap to templates folder.

  3. You can play it around with make some changes at every html files.
